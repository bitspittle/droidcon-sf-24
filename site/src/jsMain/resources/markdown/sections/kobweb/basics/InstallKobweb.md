## Install Kobweb

```bash
# Homebrew: Mac / Linux
$ brew install varabyte/tap/kobweb

# Scoop: Windows
> scoop install varabyte/kobweb

# SDKMAN! Mac / Windows / *nix
$ sdk install kobweb
```

{{{ SpeakerNotes

* You can also build yourself from source or download releases
